В компоненте app.js с помощью хука useDispatch вытаскиваем наружу нужный action creator из redux'а
в коде это выглядит как dispatch(setCount()) (в этот action надо передать то, чему соответствует этот action
оно будет хранится в качестве аргумента payload  (action.payload) в данно случае это значение count
в котором хранится число (количество)
